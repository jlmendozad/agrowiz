# AgroWiz - Market Sentinel Hub
### Applied Agentic AI for Organizational Transformation (MIT)

AgroWiz is an intelligent agent designed to provide real-time market intelligence and strategic analysis for the agricultural sector. 

## 🛠 Tech Stack
* **Language:** Python 3.10+
* **Framework:** Discord.py
* **AI Engines:** OpenAI (GPT-4) & Jina AI (Reader API)
* **Deployment:** Railway (Cloud Infrastructure)
* **CI/CD:** GitHub Integration

## 🚀 Features
* **Real-time News Extraction:** Automated web scraping for agricultural market trends.
* **Strategic Analysis:** AI-driven insights focused on digital transformation and Disagro's operational context.
* **Always-on Availability:** Cloud-hosted agent for 24/7 organizational support.
*Developed for the MIT Professional Education: Applied Agentic AI course.*
